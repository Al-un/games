# 2048 draft

Console version of 2048.

## Getting started

```sh
node ./draft/2048/index.js
```

- press any of (q, z, s, d) to move
- press "w" to exit

## Coordinates

With the following example:

```
8 0 0 0
4 0 0 0
0 0 0 2
0 0 0 0
```

`x` is the _column_ index (0 is left, 3 is right) and `y` the _row_ index
(0 is top row, 3 is bottom row):

- 8 is (0, 0)
- 4 is (0, 1)
- 2 is (3, 2)
